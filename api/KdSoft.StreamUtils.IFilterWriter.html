Interface IFilterWriter

Synchronous interface for writing data, to be used for stream-like push processing, transforming or filtering.

Namespace: KdSoft.StreamUtils
Assembly: KdSoft.StreamUtils.dll
Syntax
public interface IFilterWriter : IWriter

Methods

FinalWrite(Byte[], Int32, Int32)

Write operation, to be called at end of processing.

Declaration
void FinalWrite(byte[] buffer, int start, int count)
Parameters
Type Name Description
System.Byte[] buffer

Byte buffer containing data to be written.
System.Int32 start

Start index in buffer of data to write.
System.Int32 count

Number of bytes to write.
Remarks

After this call no more write operations are allowed. If not all data could be written, this call will throw an exception.

Parameters
Type Name Description

Byte buffer containing data to be written.
System.Int32 start

Start index in buffer of data to write.
System.Int32 count

Number of bytes to write.
Returns
Type Description
System.Int32

Number of bytes written. Can be less than requested, including 0.
Remarks

Once cannot assume that the remaining data that could not be written will be sucessfully processed on a second call. They could be insufficient for processing in one call and will need to be combined with more data.
