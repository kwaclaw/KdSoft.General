Interface IRandomAsyncWriter

Asynchronous interface for writing randomly to a data sink.

Namespace: KdSoft.StreamUtils
Assembly: KdSoft.StreamUtils.dll
Syntax
public interface IRandomAsyncWriter : IWriter
Remarks

Intended to completely fill the underlying data entity without gaps. The implementaion may, but does not have to, ignore parts of a write operation that would overlap with already written data.

Remarks
  • This can be called at any time, that is, even before other writes have been performed.
  • If the data stream already extends beyond the end position (targetOffset + count) then the implementation may, but does not have to, throw an exception. If it does not throw, then the data stream will be truncated at the end position.
  • If the completed state was reached by calling FinalWriteAsync(Byte[], Int32, Int32, TaskCreationOptions) on the same underlying entity, then this may return null even on the first call.

Remarks
  • Writing beyond the end boundaries of the writable range may result in error, this is left to the implementation. The end of the range is only known after EndWriteAsync(Byte[], Int32, Int32, Int64, TaskCreationOptions) was called.
  • The implementation may decide to consider the data complete once all data in the range was written at least once, and then return null for subsequent write requests.
