Namespace KdSoft.Serialization

The types in this namespace support the Serialization of Object Graphs.

Classes

Field<T, F>

Concrete implementations of this abstract base-class, together with a corresponding implementation of Formatter<F>, are used to serialize and deserialize an object graph.

Formatter<F>

A concrete implementation of this abstract base-class, together with corresponding implementations of ValueField<T, F> and ReferenceField<T, F> is used to serialize and deserialize an object graph.

ReferenceField<T, F>

Base field class for serializing reference types.

SerializationException

Exception class that indicates invalid serialized data, or incorrect use of the serialization framework.

ValueField<T, F>

Base field class for serializing value types.

Enums

SerialStatus

Represents the different states of a serialized field.

Delegates

AddItem<T, C>

Call-back delegate to use when adding elements to a sequence/collection that is being deserialized.

AddValueItem<T, C>

Like AddItem<T, C>, but intended for large value types.

InitSequence<T, C>

Call-back delegate to be used when deserializing a sequence.

InitValueSequence<T, C>

Like InitSequence<T, C>, but intended for large value types.
