Syntax
public class Formatter : Formatter<Formatter>
Remarks
  • The cooperating descendants of ValueField<T> and ReferenceField<T> are designed to generate serialized representations that can be used for lexical sorting.
  • The serialization API exposed by Formatter<F> is not sufficent by itself, as it contains no notion of the serialization medium - e.g. a stream or buffer. Any implementation of this API has to add medium related methods. In this case the methods are InitSerialization(Byte[], Int32), FinishSerialization(), InitDeserialization(Byte[], Int32) and FinishDeserialization(). Check the example for how they are used.
Examples

Basic usage of byte buffer oriented serialization:

byte[] buffer = new byte[2048];
StdFormatter fmt = new StdFormatter(ByteOrder.BigEndian);
new StringField(fmt, true);  // default serializer for strings
new IntField(fmt, true);     // default serializer for integers
...
int index = 0;
fmt.InitSerialization(buffer, index);
fmt.SerializeObject<string>(name);
fmt.SerializeStruct<int>(age);
index = fmt.FinishSerialization(); 
...
// do some work
...
index = 0;
fmt.InitDeserialization(buffer, index);
fmt.DeserializeObject<string>(ref name);
age = (int)fmt.DeserializeStruct<int>();
index = fmt.FinishDeserialization();

InitDeserialization(Byte[], Int32)

Initializes deserialization process.

Declaration
public void InitDeserialization(byte[] buffer, int index)
Parameters
Type Name Description
System.Byte[] buffer

Target buffer.
System.Int32 index

Start index in target buffer.
Remarks

The calls to the deserialization methods of Formatter<F> must be bracketed between calls to InitDeserialization(Byte[], Int32) and FinishDeserialization(). The same data types must be deserialized in the same order as were serialized between the original calls to InitSerialization(Byte[], Int32) and FinishSerialization(). Used together with FinishDeserialization().

Examples

In this example, SalesRep instances shared by Vendor instances were only serialized once, and thus must be deserialized the same way.

StdFormatter fmt = new StdFormatter(ByteOrder.BigEndian);
new VendorField(fmt, true);
new SalesRepField(fmt, true);
...
int index = 0;
fmt.InitDeserialization(buffer, index);
// get the field only once - more efficient than calling fmt.DeserializeObject()
VendorField vendorField = (VendorField)fmt.GetField<Vendor>();
vendorField.DeserializeObject(ref vendor1);
vendorField.DeserializeObject(ref vendor2);
vendorField.DeserializeObject(ref vendor3);
index = fmt.FinishDeserialization();

In this example, the sales rep phone number is extracted from a serialized Vendor instance, using partial deserialization by skipping nodes in the object graph. The sales rep is the 7th serialized member in Vendor, and the phone number is the 2nd serialized member in SalesRep. 

StdFormatter fmt = new StdFormatter(ByteOrder.BigEndian);
new VendorField(fmt, true);
new SalesRepField(fmt, true);
...
int index = 0;
fmt.InitDeserialization(buffer, index);
string phoneNo = "";
if (fmt.SkipMembers<Vendor>(6, 1)) {
  fmt.DeserializeObject<string>(ref phoneNo);
}
else
  throw new ApplicationException("Member path invalid.");
index = fmt.FinishDeserialization();

InitSerialization(Byte[], Int32)

Initializes serialization process.

Declaration
public void InitSerialization(byte[] buffer, int index)
Parameters
Type Name Description
System.Byte[] buffer

Target buffer.
System.Int32 index

Start index of buffer. Serialization will start writing to the buffer at this index.
Remarks

The calls to the serialization methods of Formatter<F> must be bracketed between calls to InitSerialization(Byte[], Int32) and FinishSerialization(). Any objects serialized between these two calls will have shared object references serialized only once, avoiding multiple serialization of the same object.

Used together with FinishSerialization().

Examples

In this example, SalesRep instances shared by Vendor instances are only serialized once.

StdFormatter fmt = new StdFormatter(ByteOrder.BigEndian);
new VendorField(fmt, true);
new SalesRepField(fmt, true);
...
int index = 0;
fmt.InitSerialization(buffer, index);
fmt.SerializeObject<Vendor>(vendor1);
fmt.SerializeObject<Vendor>(vendor2);
fmt.SerializeObject<Vendor>(vendor3);
index = fmt.FinishSerialization();
