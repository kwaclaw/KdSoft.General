Class ObjectEqualityComparer

Implementation of System.Collections.Generic.IEqualityComparer<T> that compares objects by reference and calculates their hash code based on the reference only. This is useful when objects need to be tracked based on their object handle even if System.Object.Equals(System.Object) and System.Object.GetHashCode() have been overridden to implement value type semantics.
