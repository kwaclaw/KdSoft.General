DeserializeValue(ReadOnlySpan<Byte>, ref T)

Specifies how and in which order struct members are deserialized.

Declaration
protected abstract void DeserializeValue(ReadOnlySpan<byte> source, ref T value)
Parameters
Type Name Description
System.ReadOnlySpan<System.Byte> source

Source to read from, a System.ReadOnlySpan<T> of bytes.
T value

The struct instance to deserialize.
Remarks

For any given member data type, this must be implemented by calling one of the overloaded versions of DeserializeObject<T>(ReadOnlySpan<Byte>), DeserializeStruct<T>(ReadOnlySpan<Byte>), DeserializeObjects<T>(ReadOnlySpan<Byte>, ref T[]) or DeserializeStructs<T>(ReadOnlySpan<Byte>, out Nullable<T>[]). The order in which members are deserialized must match the order of serialization.

Examples

Here is an example of such an implementation:

protected override void DeserializeValue(ref StockItem value) {
  Fmt.DeserializeObject<string>(ref value.name);
  Fmt.DeserializeObject<string>(ref value.category);
  Fmt.DeserializeObject<string>(ref value.vendor);
  Fmt.DeserializeObject<string>(ref value.sku);
  value.price = (float)Fmt.DeserializeStruct<float>();
  value.quantity = (int)Fmt.DeserializeStruct<int>();
}

Remarks

This method should only be called when a field instance is used directly by the application, otherwise the corresponding Formatter<F> will take care of calling it through its overloaded versions of SerializeStruct<T>(Span<Byte>, T) or SerializeStructs<T>(Span<Byte>, T[]).

Passing the value type by reference avoids copy overhead which is useful for large value types. However, this does not allow to pass nulls. To serialize a null, call SerializeNull().
