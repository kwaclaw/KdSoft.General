Struct DecimalBytes

Union used for re-interpreting a decimal as four integers and vice-versa. This works best for dealing with differences in endian-ness, since they apply only at the level of the Int32 parts. The overall storage order of these Int32 components is always: Lo->Mid->Hi->Flags.
