Returns
Type Description
System.Threading.Tasks.Task<IOResult>

A System.Threading.Tasks.Task<TResult> instance that can be waited on. The pump task will complete when all read and write tasks created as a result of this call have finished. The task result will contain the offset of the starting read and the accumulated count of all reads, as well as an indicator if the end of the data was reached.
