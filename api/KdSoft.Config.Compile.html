Type Description
System.Collections.Generic.HashSet<System.String>

Set of file names for referenced assemblies - duplicates removed.
Remarks

A 'references' section must be at the beginning of the file and each line must start with '//ref ' followed by the file name for a referenced assembly. The file name does not have to be an absolute path, as long as the compiler can find it from its current directory. A typical 'references' section looks like this:

//ref System.ServiceProcess.dll
//ref KdSoft.ServiceInterface.dll
//ref KdSoft.ServiceHost.exe
//ref KdSoft.WindowsLogon.dll
//ref KdSoft.ServiceHostIce.dll
