Class TraceExtensions

These TraceSource extensions allow creation of the trace arguments and message to be deferred until after the SourceSwitch has determined that they are needed. This has the advantage that for those trace levels that are turned off, no trace arguments will be created. This is especially an improvement for formatted strings.
