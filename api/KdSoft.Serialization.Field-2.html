Class Field<T, F>

Concrete implementations of this abstract base-class, together with a corresponding implementation of Formatter<F>, are used to serialize and deserialize an object graph.

SkipValue()

Skips serialized field value without deserializing it. Useful for partial deserialization.

Declaration
protected abstract void SkipValue()
Remarks

For a given target data type, this must be implemented by calling any of the overloaded versions of Skip<T>(Field<T, F>) for its serialized members, in the exact same order as they have been serialized. Whenever any of these calls returns false, then SkipValue() must return immediately.

Examples

Here is an example of such an implementation:

protected override void SkipValue() {
  if (Fmt.Skip<string>())
  if (Fmt.Skip<string>())
  if (Fmt.Skip<string>())
  if (Fmt.Skip<string>())
  if (Fmt.Skip<float>())
  Fmt.Skip<int>();
}
