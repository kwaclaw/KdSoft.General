DeserializeInstance(ref T)

Creates and/or initializes an instance of the field's associated type.

Declaration
protected abstract void DeserializeInstance(ref T instance)
Parameters
Type Name Description
T instance

Instance to initialize. Can be null, in which case a new instance must be created.
Remarks

May deserialize part or all of the instance - as described for DeserializeMembers(T), but must not deserialize members that may - directly or indirectly - refer back to the instance. That part of deserialization must be left to DeserializeMembers(T).

Examples

Here is an example of such an implementation:

protected override void DeserializeInstance(ref StockItem instance) {
  if (instance == null)
    instance = new StockItem();
}

DeserializeMembers(T)

Specifies how and in which order class members are deserialized.

Declaration
protected virtual void DeserializeMembers(T instance)
Parameters
Type Name Description
T instance

The object to serialize. Must not be null.
Remarks

For any given member data type, this must be implemented by calling one of the overloaded versions of DeserializeObject<T>(ref T), DeserializeStruct<T>(), DeserializeObjects<T>(ref T[], ReferenceField<T, F>) or DeserializeStructs<T>(ref T[], ValueField<T, F>). The order in which members are deserialized must match the order of serialization.

Examples

Here is an example of such an implementation:

protected override void DeserializeMembers(StockItem instance) {
  Fmt.DeserializeObject<string>(ref instance.name);
  Fmt.DeserializeObject<string>(ref instance.category);
  Fmt.DeserializeObject<string>(ref instance.vendor);
  Fmt.DeserializeObject<string>(ref instance.sku);
  instance.price = (float)Fmt.DeserializeStruct<float>();
  instance.quantity = (int)Fmt.DeserializeStruct<int>();
}
