Class AsyncBufferedReader

IRandomAsyncReader implementation that buffers a specified amount of data in a contiguous cache and allows random read requests to be satisfied from this cache.

Scheduled task instance that can be waited on, or null if reading is already complete (end of data encountered). Task result contains IOResult details.
Remarks
  • If the concurrency level exceeds the maximum, this method will not block but return the new concurrency level.
  • If the returned count is less than requested this does not mean that the end of the data was reached - see IOResult
  • However, in this implementation a read task will wait until it can read all requested data, except when it is the end of data.
